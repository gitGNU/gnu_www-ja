Utilities

bzip2 translator
When mapped on a file, you can do bzip compressed I/O. Capabilities include translating a read call, supporting SEEK_* in lseek, and dynamically determines the length of the file.
Authors: Ankur Arora , Shikka Garg and V. Venakataraman
Website: http://www.gnu-india.org/pipermail/gnu-india/20010307/000113.html

mget
Command line download manager
Author: Debajyoti Bera
Website: http://www.cse.iitk.ac.in/users/dbera/index.html

Ziplib
C++ library that allows you to read (but not write) a zip file or a jar file. The API is similar, but not indentical to java.util.zip
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/ziplib/
