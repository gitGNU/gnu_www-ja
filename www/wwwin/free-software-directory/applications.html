Applications

Bugster
P2P application for sharing MP3s and OGGs
Author: Chirag Kantharia
Website: http://sourceforge.net/projects/bugster/

Genie
Web based application to maintain genealogy databases.
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/genie/index.html

CityBus
View public transport information on the web.
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/citybus/citybus.html

Cyberoam Authentication Client
For 24hr online internet on cable service
Author: Amish Mehta
Website: http://sourceforge.net/projects/cyberoam/

httptype
Reads a list of http hosts and optionally the port number for each of these. It then queries each of the hosts and displays the HTTP server software of the host
Author: Philip S. Tellis
Website: http://httptype.sourceforge.net/
Mirror: http://staff.ncst.ernet.in/~philip/downloads/httptype/

KWireless
KDE kicker applet to display the signal quality of an IEEE 802.11b wireless network
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/KWireless/

KXML Viewer
KDE based XML viewer written in Python
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/kxmlviewer/kxmlviewer.html

MayaVI
Scientific data visualizer. Written in Python and uses the Visualization ToolKit(VTK) for the graphics. Tkinter GUI.
Author: Prabhu Ramachandran
Email: mayavi-users@lists.sf.net
Website: http://mayavi.sourceforge.net/

klookup
LDAP capable addressbook for KDE
Author: Arun Sharma
Website: http://www.sharma-home.net/~adsharma/projects/klookup.html
