CVSPermissions
CVSPermissions is a toolkit that will allow CVS administrators to setup directory level access permissions in CVS. This toolkit is a collection of shell scirpts and CVS configuration information to achieve the ACL functionality.
Author: Vivek Venugopalan
Website: http://www.magic-cauldron.com/cm/cvspermissions/cvspermissions.html
