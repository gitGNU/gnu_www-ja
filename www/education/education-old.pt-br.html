Introdução

O Software Livre pode ser um recurso valioso na educação. Ele pode não ser apenas superior técnicamente ou pedagogicamente às alternativas proprietárias, mas ele também promove os valores do Projeto GNU nas escolas:

  • Liberdade
  • Cooperação

Aqui temos algumas informações que colhemos sobre a relação do Projeto GNU com a educação (você pode ver mais relacionamentos ao ler a história do Projeto GNU). Por favor envie os seus comentários e sugestões sobre esta informação para education@gnu.org

Software Educacional no GNU

Além do GNU/Linux, do Software GNU e da Documentação do GNU, existem algums pacotes GNU criados especificamente para educação:

  • Dr. Geo é uma ferramenta gráfica e interativa que permite ao usuário explorar e descobrir a Geometria Euclidiana.
  • Ggradebook é uma aplicação gráfica para professores que permite o controle das notas dos estudantes.
  • GNU Typist é um tutor de datilografia. Ele é um interpretador para tutoriais de datilografia, que inclui tutoriais para vários idiomas e layouts de teclados.
  • Solfege é um programa para ajudar no treinamento do seu ouvido musical.
Nós mantemos uma lista de tarefas pendentes que inclui uma seção de programas relativos à educação. Você pode nos ajudar realizando uma dessas tarefas: http://www.gnu.org/prep/tasks.html#Education

Outros softwares livres úteis para educação podem ser encontrados em:

  • Na seção de Education do Diretório de Software Livre do GNU.
  • O diretório freeduc mantido pela OFSET.

Projects Educacionais no GNU

  • A Enciclopédia Livre Universal
  • O objetivo da lista info-tech-reports é avisar às pessoas sobre novos artigos, e construir uma bibliografia de relatórios técnicos livremente redistribuíveis sobre software livre.
  • O projeto OFSET é um parceito do Projeto GNU, desenvolvendo programas educacionais que são pacotes GNU.

Estatísticas

Existem alguns grupos que mantém estatísticas de escolas que utilizam software livre em várias regiões do mundo:

  • Colômbia, mantida pelo grupo SLEC.
  • França, mantida pelo grupo The Libre Software School Day.
  • Alemanha, mantida por Andreas Rittershofer.
  • Itália, mantida pelo grupo The Libre Software School Day.

Grupos e Projetos

Existem vários grupos e projetos que estão trabalhando com Software Livre e Educação. Alguns deles estão listados abaixo. Se o seu projeto suporta apenas software livre e não está listado aqui, por favor envie uma mensagem para education@gnu.org.

  • Debian Jr., promove a criação de uma distribuição de software livre apropriada para crianças
  • OFSET, promove o uso do software livre na educação.
  • SLEC, promove o uso do software livre na Colômbia.
