Only G.N.U. (And G.N.U. Alone)

(With apologies to The Platters, Buck Ram and Andre Rand)

Only G.N.U. can make this world seem free
Only G.N.U. can make the BASH a referee
Only G.N.U. and G.N.U. alone
Can thrill me like you do
And fill my ram with scripts for only you

Only G.N.U. can make this change in me
For it's true, you are my OS
When you hold off MS
I understand the source code that you do
You're my dream come true
My one and only G.N.U.

Only G.N.U. can make this change in me
For it's true, you are my destiny
When you flip on my brain
I understand the computing that you do
You're my system come free
My one and only G.N.U.
