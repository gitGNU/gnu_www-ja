• GNU/Hurd crea una lista de habla hispana.

  • Por: Nacho González
  • En: 28 de Agosto de 2002

El nuevo kernel GNU, y su grupo de usuarios y colaboradores de habla hispana han creado una lista de correo, hurd-es@es.gnu.org que servirá de punto de encuentro para dudas, opiniones, sugerencias y organizar grupos de usuarios.

Los usuarios y amigos de GNU/Hurd dispondran ahora de otro punto de encuentro aparte del canal #hurd-es de irc.gnu.org en donde debatir y sugerir ideas. Para subscribirse a esta lista se puede acceder, o bien desde el intefaz de usuario de mailman o bien mandando un mail con el asunto "subscribe" a la dirección hurd-es-request@es.gnu.org.

Hurd es un kernel basado en servidores que corren sobre un microkernel, proporcionando un sistema compatible con el estandart Unix. Este kernel esta desarrollado integramente en el proyecto GNU bajo la licencia GPL y está llamado a substituir al actual kernel Linux como nucleo oficial, en el futuro sistema GNU/Hurd o GNU a secas.
