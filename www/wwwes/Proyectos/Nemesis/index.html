¿Qué es el proyecto Nemesis? :

 

El Proyecto Nemesis consiste en un conjunto de Herramientas consistentes en Software Libre, orientadas a Internet y con entorno Gráfico. El proyecto al completo, actualmente solo está disponible para la plataforma Win32, pero en un futuro no muy lejano estarán finalizadas las versiones del mismo para las plataformas Linux, FreeBSD y otras plataformas Unix.

 

¿De qué elementos se compone el Proyecto Nemesis? :

 

NM Web Browser - Código Fuente

 

Nemesis Web Browser es un navegador basado en Internet Explorer con funciones orientadas hacia los programadores Web. Información.

 

NM WebZilla!!! - Código Fuente

 

Nemesis WebZilla Browser es un navegador basado en Gecko (Mozilla) con funciones orientadas a los programadores Web. Información.

 

NM Telnet 32

 

Cliente telnet de 32 bits que soporta terminales VT100, VT51 y ASCII. Un cliente sencillo pero funcional y que es software libre. Información.

 

X Flash Debugger

 

Es un Debugger de películas Flash. Es muy útil, por ejemplo, para programadores en Action Script . Información.
