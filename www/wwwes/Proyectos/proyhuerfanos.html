Proyectos sofware huerfanos

Estos son algunos de los proyectos software abiertos que son susceptibles de desarrollarse en la máquina es.gnu.org.
 
Para mas información acerca del hosting ofrecido, lea las normas de desarrollo de proyectos, y vea como comenzar un proyecto bajo la Coordinacion de Proyectos Software ofreciendo su adopción del proyecto. Recibirá información detallada de los servicios de hosting que ofrecemos, así como instrucciones para iniciarlo en es.gnu.org
 
  • Proyecto de mantenimiento de Goose, una librería C++ que proporciona funciones para tareas de estadística computacional. Entre sus objetivos está portar toda la funcionalidad de la biblioteca a Guile.

  • Si es un buen programador de C y le interesan los núcleos de sistemas operativos, puede contribuir en el desarrollo de GNU HURD, el kernel del sistema GNU.

  • Si le gusta Scheme, puede colaborar en el desarrollo de Guile, el lenguaje de extensibilidad del proyecto GNU. También sería deseable implementar lenguajes variados en Scheme, de manera que guile pueda hacer uso de ellos.

  • Sería muy útil disponer de un paquete de conversión de programas escritos con MS Access a Scheme, utilizando una base de datos libre, y la librería GTK.

  • Puede crear un proyecto de emulación de Windows NT sobre sistemas GNU, aunque sería deseable colaborar en alguno ya en desarrollo, como Willows Twin.


    •  
  • Puede desarrollar clones de programas de otros sistemas como MS Windows, que se ejecuten en el sistema GNU.
