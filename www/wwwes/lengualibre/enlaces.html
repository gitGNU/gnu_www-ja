Enlaces

  • Free Software Foundation: La Fundación para el Software Libre se dedica a fomentar el uso y la elaboración de software libre, es decir, un software que se puede usar, compartir y modificar libremente. La FSF ha redactado varias licencias que protegen estos derechos para todos los usuarios que utilicen software libre y, lo que nos atañe particularmente, también lo ha hecho en relación con la documentación libre. La licencia bajo la que a priori se encontrará el corpus de LenguaLibre es la FDL o Licencia de Documentación Libre.
  • GNU España: La delegación local en España del proyecto GNU (iniciado por la FSF) apoyó el proyecto prácticamente desde sus comienzos. Para conocer las actividades que llevan a cabo visite su página electrónica.
