Proprietary Jails

Here are examples of proprietary systems that are jails—they do not allow the user to freely install applications.

  • iOS, the operating system of the Apple iThings, is a jail.

  • Windows 8 on “mobile devices” is a jail.

  • Game consoles are jails, too: no game can run on the console unless the console's manufacturer has authorized it. Alas, I can't find a article to cite as a reference for this fact. Please inform us if you know of one.
