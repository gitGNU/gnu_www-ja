Traduzioni del termine ``free software''

Questo è un elenco dei termini, non ambigui, consigliati per tradurre l'inglese ``free software'' in varie lingue.

  • Arabo: Baramej Horrah
  • Bulgaro: свободен софтуер (svoboden softuer)
  • Ceco: svobodný software
  • Cinese (Semplificato): 自由软件 (zi-you ruan-jian)
  • Cinese (Tradizionale): 自由軟體 (zih-yo)
  • Coreano: 자유 소프트웨어 (ja-yu software)
  • Croato: slobodni softver
  • Danese: fri software OPPURE frit programmel
  • Ebraico: תוכנה חופשית (tochna chofshit)
  • Esperanto: libera softvaro
  • Finlandese: vapaa ohjelmisto
  • Francese: logiciel libre
  • Giapponese: 自由なソフトウェア (フリーソフトウェア - jiyuu-na software)
  • Greco: ελεύθερο λογισμικό (eleuthero logismiko)
  • Indonesiano: perangkat lunak bebas
  • Irlandese: bog earraí saoire
  • Islandese: frjáls hugbúnaður
  • Italiano: software libero
  • Lituano: laisvoji programine iranga
  • Norvegese: fri programvare
  • Olandese: vrije software
  • Polacco: wolne oprogramowanie
  • Portoghese: software livre
  • Rumeno: software liber
  • Russo: свободное программное обеспечение
  • Serbo: слободни софтвер
  • Slovacco: slobodny softver
  • Sloveno: prosto programje
  • Spagnolo: software libre
  • Svedese: fri programvara
  • Tamil: Sudanthiram software
  • Tedesco: freie Software
  • Turco: ozgur yazilim
  • Ucraino: вільне програмне забезпечення (vil'ne prohramne zabezpechennia)
  • Ungherese: szabad szoftver
  • Vietnamita: phần mềm tự do
