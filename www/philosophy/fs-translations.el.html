Μεταφράσεις του όρου ``free software''

[ Catalan | English | Greek | Japanese | Korean | Polish ]

Αυτός είναι ένας κατάλογος με αναμφίβολα προτεινόμενες αποδώσεις του όρου ``free software'' σε διάφορες γλώσσες.

  • Τσεχικά: svobodny software
  • Δανικά: fri software OR frit programmel
  • Ολλανδικά: vrije software
  • Εσπεράντο: libera softvaro
  • Φιλανδικά: vapaa ohjelmisto
  • Γαλλικά: logiciel libre
  • Γερμανικά: freie Software
  • Ελληνικά: ελεύθερο λογισμικό (eleuthero logismiko)
  • Ουγγρικά: szabad szoftver
  • Ισλανδικά:frjáls hugbúnaπur
  • Ινδονησιακά: perangkat lunak bebas
  • Ιταλικά: software libero
  • Ιαπωνικά: jiyuu-na software
  • Κορεάτικα: ja-yu software
  • Νορβηγικά: fri programvare
  • Πολωνικά: wolne oprogramowanie
  • Πορτογαλικά: software livre
  • Slovak: slobodny softver
  • Slovenian: Prosto programje
  • Ισπανικά: software libre
  • Σουηδικά: fri programvara
  • Τουρκικά: ozgur yazilim
