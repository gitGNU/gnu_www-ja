Traducciones de la expresión «free software»

La siguiente lista enumera la traducción a varios idiomas que recomendamos usar para la expresión inglesa «free software» con la finalidad de evitar ambigüedades:

  • Alemán: freie Software
  • Árabe: Baramej Horrah
  • Bielorruso: свабоднае праграмнае забесьпячэньне (svabodnaye pragramnaye zabespyachenne)
  • Búlgaro: свободен софтуер (svoboden softuer)
  • Catalán: programari lliure
  • Checo: svobodný software
  • Chino simplificado: 自由软件 (zi-you ruan-jian)
  • Chino tradicional: 自由軟體 (zih-yo)
  • Coreano: 자유 소프트웨어 (ja-yu software)
  • Croata:slobodni softver
  • Danés: fri software (o también «frit programmel»)
  • Eslovaco: slobodny softver
  • Esloveno: prosto programje
  • Español: software libre
  • Esperanto: libera softvaro
  • Finés: vapaa ohjelmisto
  • Francés: logiciel libre
  • Griego: ελεύθερο λογισμικό (elefthero logismiko)
  • Hebreo: תוכנה חופשית (tochna chofshit)
  • Hindi: मुक्त सॉफ्टवेयर (mukt software)
  • Holandés: vrije software
  • Húngaro: szabad szoftver
  • Indonesio: perangkat lunak bebas
  • Irlandés: bog earraí saoire
  • Islandés: frjáls hugbúnaður
  • Italiano: software libero
  • Japonés: 自由なソフトウェア (フリーソフトウェア - jiyuu-na software)
  • Lituano: laisvoji programine iranga
  • Noruego: fri programvare
  • Persa: نرم‌افزارِ آزاد
  • Polaco: wolne oprogramowanie
  • Portugués: software livre
  • Rumano: software liber
  • Ruso: свободное программное обеспечение
  • Serbio: слободни софтвер
  • Sueco: fri programvara
  • Tamil: Sudanthiram software
  • Turco: õzgür yazilim
  • Ucraniano: вільне програмне забезпечення (vil'ne prohramne zabezpechennia)
  • Vietnamita: phần mềm tự do
  • Zulu: Isoftware Ekhululekile
