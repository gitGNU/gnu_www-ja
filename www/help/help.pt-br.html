Guia do Projeto GNU

Os coordenadores dos voluntários do GNU <gvc@gnu.org> podem ajuda-lo se você deseja ajudar o Projeto GNU. Eles podem coloca-lo em contato com outras pessoas interessadas em trabalhar em projetos similares. Quando você selecionar uma tarefa da nossa lista de tarefas, por favor comunique a eles que você está interessado em trabalhar nessa tarefa.

Por favor envie questões sobre o Projeto GNU para gnu@gnu.org, comentários sobre estas páginas web para webmasters@gnu.org.

Como ajudar o Projeto GNU

  • Escreva, avalie e use software livre.

    • Instale e use software livre. A maneira mais fácil de fazer isso no momento é instalar e utilizar um sistema GNU/Linux.

    • Avalie software livre.
      • Você pode se tornar um membro do Grupo de Avaliação de Software GNU.

    • Escreva software livre.
      • A página Lista de tarefas GNU documenta novas tarefas que necessitam de voluntários.
      • A página de ajuda do Savannah contém pedidos de ajuda para projetos do GNU que estão hospedados em savannah.
      Quando escrevendo software, por favir siga os Padrões GNU de Codificação e leia Informações Para Mantenedores de Software GNU à medida que você mantém o software

      Se você necessita de algum recurso para desenvolver software GNU, por favor veja a página de Recursos para o Desenvolvedor de Software GNU.

  • Escreva documentação para o software GNU, utilizando os seguintes recursos, dicas e conselhos.

  • Por favor ajude escrevendo melhores descrições dos livros GNU publicados.

  • Seja voluntário como revisor assinando a lista de e-mail dos revisores (você faz isso enviando uma mensagem contendo no seu corpo a palavra "subscribe" para <proofreaders-request@gnu.org>)

  • Seja voluntário para ajudar na compilação de um diretório de todo o software livre conhecido. Esta tarefa envolve baixar pacotes de software livre, sumarisar informações sobre estes pacotes, e verificando as licensas dos programas.

    Se você está interessado em ser voluntário para esta tarefa, por favor contacte <bug-directory@gnu.org<.

  • Seja voluntário para garantir que os artigos em nossa página de filosofia e outras URLs do site do GNU estejam ou sejam linkadas por diretórios da internet, portais, e vários índices hierárquivos da WWW como o Yahoo!, dmoz.org e o Google. Faça com que esses sites adicionem registros detalhados sobre nossas diferentes páginas. Tenha certeza de que os artigos em nossa página de filosfia e outras URLS do site do GNU sejam listadas nas categorias apropriadas.

    Se você gostaria de nos ajudar nesta tarefa, por favor contacte os Coordenadores de Voluntários GNU <gvc@gnu.org>.

  • Fale a outras pessoas sobre o Projeto GNU e a Fundação para o Software Livre:

    • Informando os seus amigos sobre a filosofia e sobre o software GNU.

    • Na sua home-page, adicione um link para a página principal do GNU,
      e peça a outros para fazerem o mesmo.

    • Espelhe este site web, assim como outros sites já estão fazendo,
      e peça a mais pessoas para fazerem o mesmo.

  • Ajude a FSF a levantar fundos para nos ajudar a escrever mais software livre e mais documentação:

    • Fazendo uma doação para a FSF.

    • Comprando manuais, camisetas e principalmente CD-ROMs da FSF sempre que você possa. A maioria da renda da FSF vem da venda de cópias de coisas que todos são livres para copiar.

    • Convença a sua empresa a comprar manuais, camisetas e principalmente CD-ROMs sempre que você possa.

    • Convença a sua empresa a comprar uma Distribuição de Luxo da FSF.

    • Venda software livre e doe uma parte da renda para a Fundação para o Software Livre ou outro projeto de desenvolvimento de software livre. Financiando o desenvolvimento, você pode ajudar o mundo do software livre a ir adiante.

      Distribuir software livre é uma oportunidade de levantar fundos para o desenvolvimento. Não a desperdice!

  • Doe manterial de escritório e equipamentos para a FSF. Ofereça as doações para <gnu@gnu.org>.

  • Assuma uma das tarefas (14k caracteres) que nós necessitamos neste servidor web.

  • Se você ou sua empresa suportam o desenvolvimento de software livre de alguma forma, você pode ser listado (ou sua empresa) no Diretório de Serviços GNU.

  • Se você gerencia uma empresa que necessita contratar pessoas para trabalhar com software livre, você pode anunciar em nossa Página de Empregos com Software Livre.
