Informações para a Imprensa sobre a FSF e o GNU

[ Inglês | português do Brasil ]

Esta página é o local para informações relacionadas com a imprensa sobre a Fundação para o Software Livre e sobre o Projeto GNU. Nós mantemos arquivos de nossos press releases aqui, e outras informações de interesse para o pessoal da mídia.

Lista de Mensagens para a Imprensa

Se você está interessado em informações relacionadas com a imprensa sobre a Fundação para o Software Livre e sobre o Projeto GNU, você pode assinar a lista info-press seguindo este link ou enviando uma mensagem para <info-press-request@gnu.org> contendo a palavra "subscribe" no corpo da mensagem.

Arquivos da Lista info-press estão disponíveis aqui.
