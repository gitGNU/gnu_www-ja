Use 'GNU

Expresia ``GNU's Not Unix'' este arcuită deasupra, iar ``Free Software Foundation'' mărginește dedesubt această instrucțiune tip Lisp despre GNU. Două ``parenteze'' formate de GNU-uri curbate, dansând formează un program pseudo-Lisp: (USE 'GNU).

Parantezele ce dansează au fost desenate de Etienne Suvasa. Ele apar pe coperta cărții GNU Emacs Lisp Reference manual.

Restul acestei lucrări a fost designul personalului FSF pentru tricoul GNU din 1996.
