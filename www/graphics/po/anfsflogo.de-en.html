An FSF Logo

Two similar, rectangular, unofficial logos for the Free Software Foundation. FSF is written in red, block-style lettering, with a string of 0's and 1's in the background.

We thank Shawn Ewald <ewald@ctaz.net> for designing these FSF logos.

He prefers the one with the black background, so we used to use that on the FSF page.

This picture is available in the following formats:

  • jpeg 8k (black background), 6k (white background)
  • png 1k (black background), 1k (white background)
