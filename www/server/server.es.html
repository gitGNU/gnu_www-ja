Acerca de www.gnu.org

Aquí se encuentra una lista de sitios espejo (mirrors) de este sitio web, y unas notas sobre cómo realizar un espejo de este sitio web.

Para el diseño y creación de nuestra web, seguimos nuestra guía de estilo. Si desea colaborar, tenemos una lista de tareas que necesitan realizarse en nuestro sitio web.

Todas nuestras páginas se ven bien con cualquier navegador (campaña "Best Viewed with Any Browser").

Ya que no instalamos software privativo, tenemos la política de no enlazar con páginas que promuevan programas privativos. No podemos hacer que el software privativo desaparezca si no hablamos de él; la información sobre software privativo está ampliamente disponible, y la gente que lo quiera lo conseguirá. Pero preferimos no participar en el proceso de anunciar software privativo.

Además confinamos en unas pocas páginas todos los enlaces a textos con carácter comercial: en aquellas páginas cuyo contenido hace los enlaces apropiados. De este modo tratamos de invertir la creencia extendida de que el dinero y los negocios ocupan un alto lugar en la vida.

En esta computadora sólo se emplea software libre. El servidor web es el servidor web Apache, el cual se distribuye bajo la licencia Apache. Para mantener este servidor, empleamos diverso software libre desarrollado por los voluntarios de GNU y además mantenemos una lista con este tipo de software.

Estamos hospedados actualmente a través de Global NAP, que nos ha donado el ancho de banda. El sistema fue donado por HP. Para una información más detallada, por favor, lea nuestra configuración del servidor.

El sistema operativo es Debian GNU/Linux, el cual es un ejemplo de un sistema GNU/Linux.
