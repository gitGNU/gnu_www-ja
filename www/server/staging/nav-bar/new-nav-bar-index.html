Drafts for the New Navigation Bar and General Website Improvement

We have received feedback from visitors about the need to improve website navigation (“make it easier to find your way around”) and the overall look and feel of it. ( RT #769591 Sharpening up gnu.org.) Here are some attempts to achieve this.

Some past discussions on the subject for reference and to get an idea of what people would like to achieve:

Improving Site Navigation.
Finding Translations.
Organizing the Philosophy Section.
More on Navigation (/philosophy).
Improving Readability.
Where to place “Join the FSF!”.
Redesigning gnu.org.

Important notes

  • The design of the gnu.org website should follow the W3C accessibility guidelines. For more on this, see our accessibility recommendations.
  • It has been agreed that the list of translations should be at the top of the page.
  • There is a strong preference that the call to “join the FSF” be placed in a prominent area of the page.

Drafts

Navigation Bar

The following are attempts to improve navigability. Click and hover on the navigation bar to see how it behaves (not all items are active.) Discussions about these tests can be found at https://lists.gnu.org/mailman/private/www-discuss/2011/007429.html and https://lists.gnu.org/mailman/private/www-discuss/2012/007483.html.

  1. First attempt.
  2. test7. A solution to keep the sub-nav lined up under the main navigation bar. Try to make selected items more eye-catching by using a bigger font. Make #joinfsftab text become blue when selected to avoid low contrast ratio.
  3. test7-inner Another solution to keep the sub-nav lined up under the main navigation bar.
  4. test7-bkg-color Make items in the sub-nav more eye-catching by introducing a background color instead of modifying the font size.
