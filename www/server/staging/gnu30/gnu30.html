Celebrate with us

Thirty years ago this month, the GNU system announcement sparked a conversation that has grown into the global free software movement. Now we invite you to join the GNU community in celebrating this important occassion, and creating a future where GNU is stronger than ever.

Get started by taking action with GNU-a-Day, signing up for updates or exploring the celebration events around the world.

GNU-a-Day

Celebrate GNU's 30th anniversary with GNU-a-Day, 30 software freedom actions you can take in the month of September.

Day 1: Read the GNU Manifesto, and share it with your networks using the hashtag #gnu30.

Day 2: Friends don't let friends use Windows 8. Pledge to switch to GNU/Linux or to help a friend switch.

Day 3: Start using LibreOffice and OpenDocument instead of Microsoft Office.

Day 4: Submit a program to the Free Software Directory or find a program in it and download it.

Day 5: Switch to a free software pdf reader.

Day 6: Submit a bug report or feature request for a program you use.

Day 7: Add your hardware to H-Node hardware database.
