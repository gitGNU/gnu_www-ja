Introducción

Aquí está la lista de directivas a seguir por los voluntarios de la web de GNU que trabajen en http://www.gnu.org/

Para averiguar quién está al cargo de qué parte de www.gnu.org, compruebe el mapa. Para más información sobre el mapa vea la organización de www.gnu.org

Archivos "Léeme" varios

  • Algunas directivas para webmasters de GNU.
  • La hoja de estilo HTML de la FSF, http://www.gnu.org/server/fsf-html-style-sheet.html
  • Información sobre cómo ser webmaster de www.gnu.org
  • Guía para traducir las webs de GNU a otros lenguajes.
  • Directivas para la creación de páginas web en www.gnu.org
  • Guía de administración de Savannah, el clon de SourceForge dedicado al Proyecto GNU.
  • Cómo crear o actualizar webs en http://www.gnu.org/software/ para mantenedores o voluntarios.
  • Aquí está la ayuda (14.000 caracteres) que necesitamos con nuestro servidor web.
  • Si usted está interesado en duplicar esta web, por favor lea la Guía de duplicación.
  • La configuración del servidor web de GNU.
