Espelhos do www.gnu.org

[ Coreano | Inglês ]

Aqui esta uma lista, por país, dos espelhos do www.gnu.org, o server primario da World Wide Web do Projeto GNU foi patrocinado pela Free Software Foundation (FSF). Aqui esta a lista dos nossos espelhos de FTP .

Se você gostaria de criar um espelho desse site, por favor leia os conselhos(vou mudar isso), e depois fale ao webmasters@gnu.org par adicionar você a esta lista quando você tiver criado um espelho desse site.

O Web site primario do Projeto GNU esta razoavelmente ocupado. Por favor, use um dos espelhos do nosso Web sever primario que estiver mais perto de você.

Este são os paises que possuem espelhos:
