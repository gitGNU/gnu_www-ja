歡迎來到 GNU 計畫的網頁伺服主機 www.gnu.org 。 GNU 計畫 開始於1984年， 旨在發展一個 類-Unix 且為 自由軟體 的完整作業系統： GNU 系統。 （GNU 是由“GNU's Not Unix”所遞迴定義出的首字母縮寫語； 它的發音為“guh-NEW”）。 各種使用 Linux 作為內核的 GNU 作業系統正被廣泛地使用著； 雖然這些系統通常被稱作為“Linux”， 但是它們應該更精確地被稱為 GNU/Linux 系統 。

這裡同樣也是自由軟體基金會的網站（FSF）。 FSF 是 GNU 計畫的主要贊助組織。 FSF 只由公司或作出轉讓的基金會處收到很少的資金。 我們依賴來自於像您這樣， 支持 FSF 保存、保護以及促進「使用、研究、複製、修改和重新散佈電腦軟體」， 並且保衛著自由軟體使用者的權利的任務的個人的支持。 去年， 我們的運作資金超過百分之六十七來自於個人的捐贈。 持續進行的支持是我們可以繼續我們的工作的主要方式。 請考慮 今天就作出捐贈 ， 或是成為 FSF 的準成員 。 您也可以訂購理查•史托曼的選文書籍 《自由軟體，自由社會》 。

GNUs 短訊（Flashes）

一個有關於 在您所在地區的書局取得 GNU Press 書籍 的新頁面已被加入。

W3C “免版稅”專利權提案並沒有保護自由軟體社群， 以使其可以完全參與「網路標準的實作與擴充」的權利。 請閱讀更多有關於這個議題， 並將您的意見送到 W3C 。

發表理查•史托曼〔所撰寫〕的最新書籍 - 《自由軟體，自由社會：理查•馬修•史托曼的選文》 ，由 Lawrence Lessig 撰文介紹。這個選文集合涵蓋了許多廣泛的議題，從 GNU 計畫 的歷史、社群的建立到 DMCA 〔所造成〕的危險。

這裡有份 GNU 計畫 最新發行軟體 的列表。 我們每晚都更新。

對於其他消息以及先前放在 GNUs 短訊 的項目， 請見在 GNU 計畫中或與其有關的 最新消息 。

採取行動

緊急通告： 請參閱 http://www.digitalspeech.org 並且幫忙阻止媒體公司想要強加到您的電腦， 以限制它所能處理的事情上的運動。

給歐洲公民的特急通報： 歐洲委員會正在推動使所有的〔創意〕想法都具有專利權， 提案已由 BSA 寫成。 此頁 解釋了一切相關訊息並且告訴您可以如何幫忙以避免發生這個大災難。

美國公民們： 請到 對抗軟體專利權請願書 以及 反-DMCA 請願書 處簽名。

更多資訊

自由軟體基金會 是一個致力於募集資金， 以使 GNU 計畫得以進行的的免稅慈善機構。

GNU 計畫的哲學和歷史可在 理查•史托曼 的文章、 《GNU 計畫》 以及許多其他放在我們的 哲學 章節中的文章找到。

自由軟體基金會支持在網路上的 言論、出版以及集會結社自由 、 在私人通信上使用加密軟體的權利 以及不受私人壟斷所阻礙的 軟體編寫權利 。
