 System compatibility:
  • If you are using GNU C++, you need at least version 2.8.1.
  • It should build with egcs version 1.0 and later.
  • The system should support ELF binary format in order to be able to use plugins. If ELF format is not supported, you can try building a statically linked binary (plugins will be statically linked to main code).
  • To build the GUI, you need to have GTK+ and GTK--. It has been tested with GTK-- version 1.0.0. It may need some modifications to work with other versions.
