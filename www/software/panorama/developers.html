Notes for developers

  If you think you can help with this project, you should start by reading the project list. You should also read all files in 'doc/developers' directory, in the source distribution.

  If you have a good knowledge of GTK+/GTK-- and you want to continue with the graphic interface, the project is yours. I currently don't have time to keep with it. To see how this graphic interface looks like, take a look at this screenshot.

  We also need people to write some documentation (tutorial / reference). You can see a simple draft in 'doc/users'.

  In the future, I hope I can fill this page with links to pages of people making plugins for Panorama.
