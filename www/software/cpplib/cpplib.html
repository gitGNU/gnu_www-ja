C++ Library

The GNU libstdc++ library implements the library facilities defined by the forthcoming ISO C++ standard. This includes strings, iostream, and various container classes. All of this is templatized.

Additional non-standard routines are available in the separate libg++ distribution, but that library is no longer supported, and new programs should avoid using it.
