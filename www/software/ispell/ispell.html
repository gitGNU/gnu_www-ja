Ispell

Ispell is a program that helps you to correct spelling and typographical errors in a file. When presented with a word that is not in the dictionary, ispell attempts to find near misses that might include the word you meant.

We use Ispell as part of the GNU system but Ispell itself is not a GNU program. These days, you might like to look at GNU Aspell.

Downloading Ispell

Ispell can be found on in the subdirectory /non-gnu/ispell/ on your favorite GNU mirror. For other ways to obtain Ispell, please read How to get GNU Software
