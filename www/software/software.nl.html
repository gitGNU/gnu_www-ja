Download distributies

Wanneer je op zoek bent naar een compleet te installeren systeem, zie dan onze lijst van GNU/Linux distributies die geheel uit vrije software bestaan.

Wanneer je op zoek bent naar vrije software pakketten, zowel GNU als niet-GNU, kijk dan naar de Free Software Directory: een doorzoekbare databank, in categorieën ingedeeld, van vrije software. De data wordt bijgehouden door de Free Software Foundation en bevat verwijzingen naar homepages als die bekend zijn. Er zit ook een lijst in van alle GNU pakketten met hun documentatie en andere informatie.

We houden ook een lijst bij van vrije software projecten met een hoge prioriteit. Help deze projecten alsjeblieft wanneer je in de gelegenheid bent. Zie GNU helpen, voor andere manieren om GNU te helpen.

Wanneer je een GNU pakket aan het ontwikkelen bent of hierover denkt, kijk dan eens bij GNU Software Hulpmiddelen voor Ontwikkelaars. Wanneer je erover denkt van je pakket een officieel GNU pakket te maken, kijk dan op evaluatie informatie en aanbiedingsformulier.

Voor meer informatie over de licenties die GNU software gebruikt en andere licenties, zie onze Licenties pagina.

Tenslotte is er een korte lijst met vrije software voor Microsoft Windows, voor de Windows gebruiker die benieuwd is naar vrije software.

Hoe Verkrijg Ik GNU Software

GNU software is op diverse manieren beschikbaar:

  • Koop een distributie van de FSF. Hiermee sponsor je het GNU Project.
  • Download via web of ftp: we leveren broncode voor alle GNU software, zonder bijkomende kosten. (Geef daarbij alsjeblieft een donatie aan de FSF, wanneer je het kan missen, zodat we meer vrije software kunnen maken.)
  • Kopieer het van je vrienden. (Geef daarbij alsjeblieft een donatie aan de FSF, wanneer je het kan missen, zodat we meer vrije software kunnen maken.)
  • Koop een computer met een compleet vrij GNU/Linux besturingssysteem voorgeïnstalleerd, van één van deze bedrijven die dit in het assortiment hebben.
  • Haal de laatste versie van de broncode van een pakket op en bouw het zelf. Zo kun je gelijk alle laatste snufjes uitproberen. Veel GNU pakketten onderhouden hun broncode op de GNU hosting site savannah.gnu.org . Sommige pakketten gebruiken andere opslag methodes voor broncode: de webpagina van het pakket zal daar veelal naar verwijzen.

Uitgefaseerde GNU pakketten

Soms worden GNU pakketten uitgefaseerd, meestal omdat ze zijn opgevolgd door een nieuw pakket of opgenomen in een ander, groter pakket.
