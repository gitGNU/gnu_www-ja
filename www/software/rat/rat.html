Recipe Analysis Tool

The rat package was decommissioned in 2008, since the gnutrition package supersedes it.

The Recipe Analysis Tool (RAT) is a program to calculate the total nutrients in a given recipe, using an American Government database. Its features includes:

  • both Imperial and metric weights supported;
  • a wide range of nutrients, including trace minerals and amino acids;
  • can output in LaTeX format;
  • database has over 5,000 entries;
  • calculates recommended daily intakes;
  • recipes are written as text files and then "compiled" to produce the nutrient information;
  • Recommendet Daily Intakescan be calculated for many nutrients.
Note that rat is still in development and may have bugs. You have been wraned.
