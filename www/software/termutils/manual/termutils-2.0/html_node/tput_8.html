Options

The options supported by GNU tabs are:

-T termtype
--terminal=termtype
Specify the terminal type, overriding the environment variable TERM.
-V
--version
Show the version number of GNU tabs and exit.
-h
--help
Show the command line usage of GNU tabs and exit.
