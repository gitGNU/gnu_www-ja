2.6.4 Raw graph options

The following option is relevant only to raw graph, i.e., is relevant only if no output format is specified with the ‘-T’ option. In this case graph outputs a graphics metafile, which may be translated to other formats by invoking plot. This option should appear on the command line before any file names, since it affects the output of the plot (or multiplot) as a whole.

‘-O’
‘--portable-output’
Output the portable (human-readable) version of GNU metafile format, rather than a binary version (the default). This can also be requested by setting the environment variable META_PORTABLE to "yes".
