Appendix G Reporting Bugs

Please report all bugs in the GNU plotting utilities, including the libplot library, to bug-plotutils@gnu.org. Be sure to say which version of the plotting utilities package you have. Each command-line program announces the package version if you use the ‘--version’ argument.

If you installed the package from scratch, be sure to say what compiler (and compiler version) you used. If your problems are installation-related, be sure to give all relevant information.
