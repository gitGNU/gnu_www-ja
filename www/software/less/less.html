GNU less

GNU less is a program similar to more, but which allows backward movement in the file as well as forward movement. Also, less does not have to read the entire input file before starting, so with large input files it starts up faster than text editors like vi. Less uses termcap (or terminfo on some systems), so it can run on a variety of terminals. There is even limited support for hardcopy terminals.

Downloading less

less can be found on the main GNU ftp server: http://ftp.gnu.org/gnu/less/ (via HTTP) and ftp://ftp.gnu.org/gnu/less/ (via FTP). It can also be found on the GNU mirrors; please use a mirror if possible.

Mailing lists

less uses the following email for bug reports: <bug-less@gnu.org>

Announcements about less and most other GNU software are made on info-gnu (archive).

Security reports that should not be made immediately public can be sent directly to the maintainer. If there is no response to an urgent issue, you can escalate to the general security mailing list for advice.

Getting involved

Development of less, and GNU in general, is a volunteer effort, and you can contribute. For information, please read How to help GNU. If you'd like to get involved, it's a good idea to join the discussion mailing list (see above).

Maintainer
less is currently being maintained by Mark Nudelman. Please use <bug-less@gnu.org> for contact.
