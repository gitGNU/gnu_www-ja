Introduction to Patch

`patch' takes a patch file containing a difference listing produced by diff and applies those differences to one or more original files, producing patched versions.

Downloading Patch

Patch can be found on in the subdirectory /gnu/patch/ on your favorite GNU mirror. For other ways to obtain Patch, please read How to get GNU Software
