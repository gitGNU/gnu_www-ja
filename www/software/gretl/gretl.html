gretl

gretl is an econometrics package comprising a shared library, a command-line client program and a gui client which uses GTK+. It comes with many sample data files and a database of US macroeconomic time series. It supports several least-squares based statistical estimators but does not at present offer maximum likelihood methods. It calls gnuplot to generate graphs and is capable of generating output in LaTeX format. As of this writing gretl is under development: expect some bugs. For more information, see the gretl home page.
