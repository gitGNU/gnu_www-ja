obst

OBST was originally designed to serve as the common persistent object store for the tools in software engineering environments. The FSF no longer distributes OBST; as far as we know, it has not been maintained for many years, and people no longer use it. If this is not true, please contact us with new info about who is maintaining or using OBST.
