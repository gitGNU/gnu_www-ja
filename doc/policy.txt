1. 用語

用語については、

    dict/word.ja.po

を参照ください。

必要に応じて足して行ってください。


2. 用字

句読点は: 、 。
を用いています。
ただし、Keyword のところは機械処理をされるので , (カンマ)を使っています。

日本語での quote は、「」と『』を使うようにしています。
英語の quote をそのまま用いるときは、「」と『』にはせず、元の quote を使う。 

もとは英語の外来語の単語間の区切りは・を使う。


3. Wikipedia を参照する URL

該当ページの日本語版の方が読者にとってとても良いと言うものでない限り、
原文のものをそのまま用いてください。


4. ページの著作権表示の Creative Commons

各言語で書くようにというのが GNU Project の指示ですが、日本語の場合、Creative Commons
の現行の(as of 2012)翻訳が変で、"Attribution-NoDerivs"を「表示-禁止」と意味不明に
訳しているので(以前の 2.x はわかりやすかった)、日本語ではなく、英語そのままとしてあります。
FSF の了解はとってあります。


A. これまでとの違い

- 各国語訳 -> 各言語訳
- 哲学 -> 理念
- Free Software   自由ソフトウェア
- free software   (文脈によって) 自由なソフトウェア
- Free Software Foundation フリーソフトウェアファウンデーション   (財団と言わない)
- non-free software 不自由なソフトウェア
- non-free system distribution 不自由システム・ディストリビューション
- proprietary software プロプライエタリ{な,の}ソフトウェア

- 単語
** version         版     バージョン も OK 
** modification    改変 変更
** change          変更
** copy            コピー  複製 もOK
** distribute      配布   (頒布と言わない)
** redistribute    再配布  (再頒布と言わない)
- あんまり漢字を使いすぎない
